- Analyze numerical datasets efficiently  
- Perform statistical computations  
- Identify trends and patterns

---

## 📚 Theory

- 🔹 **Numerical Data**
  - Represents measurable values  
  - Examples include marks, age, income, scores  

- 🔹 **Descriptive Statistics**
  - Provides a summary of dataset characteristics  
  - Includes measures of central tendency and dispersion  

- 🔹 **Mean (Average)**
  - Indicates the overall average of the dataset  

- 🔹 **Median**
  - Represents the middle value in sorted data  

- 🔹 **Mode**
  - Shows the most frequently occurring value  

- 🔹 **Standard Deviation**
  - Measures how spread out the data is  

- 🔹 **Variance**
  - Indicates variability within the dataset  

- 🔹 **Pandas Functions**
  - Used for quick statistical analysis  
  - Examples: `mean()`, `median()`, `mode()`, `describe()`  

- 🔹 **NumPy**
  - Optimized for numerical and mathematical operations  

- 🔹 **Data Cleaning**
  - Ensures accuracy by handling missing or inconsistent values  

- 🔹 **Sorting & Filtering**
  - Helps in better interpretation and analysis
