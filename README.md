## 🔰 Introduction

Data wrangling (also known as data preprocessing) is the process of **cleaning, transforming, and organizing raw data** into a structured format suitable for analysis.

In real-world scenarios, datasets are often:
- Incomplete  
- Inconsistent  
- Noisy  

This experiment focuses on using Python libraries to prepare data for analysis by applying various wrangling techniques.

---

## 🎯 Objectives

- Clean and preprocess raw datasets  
- Handle missing and inconsistent values  
- Transform data into usable formats  
- Improve data quality for analysis  
- Perform efficient data manipulation using Python  

---

## 📚 Theory

- 🔹 **Data Wrangling**
  - The process of converting raw data into a clean and usable format  

- 🔹 **Missing Data Handling**
  - Removing missing values using `dropna()`  
  - Filling missing values using `fillna()`  

- 🔹 **Data Cleaning**
  - Removing duplicates using `drop_duplicates()`  
  - Fixing inconsistent data entries  

- 🔹 **Data Transformation**
  - Changing data types using `astype()`  
  - Applying functions using `apply()`  

- 🔹 **Renaming Columns**
  - Improves readability using `rename()`  

- 🔹 **Filtering Data**
  - Extracting relevant data using conditions  

- 🔹 **Sorting Data**
  - Arranging values using `sort_values()`  

- 🔹 **Indexing & Selection**
  - Accessing data using `loc[]` and `iloc[]`  

- 🔹 **Feature Engineering**
  - Creating new columns based on existing data  

- 🔹 **Handling Outliers**
  - Identifying abnormal values  
  - Removing or transforming them
